用接口模拟可伸缩的枚举
用一个枚举类型去扩展另一个枚举类型不是一个好的做法，而应该用枚举类型扩展接口的方式。
BasicOperation.java
如果还要定义一个BasicOperation.java中的扩展，所要做的就是编写一个枚举类型，让它实现Operation接口。
ExtendedOperation.java
在可以使用基础操作的任何地方，都可以使用新的操作，只要API是被写成采用接口类型而非实现。
用接口模拟可伸缩枚举有个小小的不足，即无法将实现从一个枚举类型继承到另一个枚举类型。
总之，虽然无法编写可扩展的枚举类型，却可以通过编写接口以及实现该接口的基础枚举类型，对它进行模拟。这样允
许客户端编写自己的枚举来实现接口。如果API是根据接口编写的，那么在可以使用基础枚举类型的任何地方，也都可
以使用这些枚举。
